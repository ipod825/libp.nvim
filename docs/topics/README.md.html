Installation

Use you preferred package manager. Below we use packer.nvim as an example.

use {'nvim-lua/plenary.nvim'}
use {'ipod825/libp.nvim'}

or

use {'ipod825/libp.nvim', requires={'nvim-lua/plenary.nvim'}
