Usage:

range (beg[, ends=beg[, step=1]])
Returns a VIter iterating through a range with some step.

Parameters:

  • beg number The begin of the range. If ends is omitted, begin becomes 1 and ends is set to the value of beg
  • ends number The end of the range (default beg)
  • step number The incremental on each iteration (default 1)

Returns:

Usage:

    assert.are.same({ 1, 2, 3 }, itertools.range(3):collect())
assert.are.same({ 3, 2, 1 }, itertools.range(3, 1, -1):collect())
