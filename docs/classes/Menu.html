Class Menu

Module: libp.ui.Menu

Menu class.

A ui for user to pick entry and execute corresponding actions. Note that Menu has dedicated API supporting plenary async context. The following is callback style non-async usage

local m = Menu({ content = { "a", "b", "c" } }) m:show()
