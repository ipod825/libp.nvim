Class KVIter

Module: libp.datatype.KVIter

Value type iterator.

An implementation of the Iter interface iterating over the key/value pair(s) of the underlying container or generator function.

Inherits: Iter

Methods

KVIter:collect () Returns a table hosting the results of next calls (until it returns nil).
KVIter:next () Returns the key/value pair of the kv iterable (see Iter) and moves the iterator to the next position.

KVIter:collect ()
Returns a table hosting the results of next calls (until it returns nil).

Returns:

    table

Usage:

    assert.are.same({ a = 1, b = 2 }, KVIter({ a = 1, b = 2 }):collect())
KVIter:next ()
Returns the key/value pair of the kv iterable (see Iter) and moves the iterator to the next position. This function is triggered by the __call operator and is thus for-loop compatible. However, users can also calls it explicitly to get just the next result.

Usage:

  • for i, v in KVIter({ 1, 2, 3 }) do
    assert(i == v)
end
  • local iter = KVIter({ a = 1, b = 2 })
assert.are.same({ "a", 1 }, { iter:next() })
assert.are.same({ "b", 2 }, { iter:next() })
assert(iter:next() == nil)
