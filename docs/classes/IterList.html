Class IterList

Module: libp.datatype.IterList

List iterator class.

A List can be transformed to IterList by List.to_iter. While List already provides some functional programming style functions such as List.filter and List.map, IterList makes it more general such that the filter and map operations can be chained. In fact, the List APIs were provided to reduce verbosity in simple cases. Those APIs simply call IterList APIs.

IterList.from_range (beg[, ends[, step=1]])
Creates an IterList from a range.

Parameters:

  • beg number The start of the range (included). If ends is not provided, beg specifies the ends and beg is implicitly set to 1.
  • ends number The end of the range (included) (optional)
  • step number The step for each iteration. Can be negative (default 1)

Returns:

    IterList The IterList representing the range.

Usage:

    assert.are.same({ 1, 2, 3, 4 }, IterList.from_range(4))
assert.are.same({ 1, 3, 5 }, IterList.from_range(1, 6, 2))

IterList:for_each (fn)
Executes a function on each element of the list.

Parameters:

  • fn function(any)->nil The function to be executed

Returns:

    nil

Usage:

    local sum = 0
local indices = {}
IterList.from_range(1, 4):for_each(function(e, i)
    sum = sum + e
    table.insert(indices, i)
end)
assert.are.same(10, sum)
assert.are.same({ 1, 2, 3, 4 }, indices)
