List:__index (indices)
Returns a slice of the original list.

Parameters:

  • indices array The indices of the slice, both inclusive. The first element is the begin of the slice and the second is the end of the slice. If first is nil, then it defaults to 1. If the second is nil, then it defaults to the length of the array.

Returns:

    List A new List

Usage:

    local lst = List({ 1, 2, 3, 4 })
assert.are.same(2, lst[2])
assert.are.same({ 2, 3, 4 }, lst[{ 2 }])
assert.are.same({ 2, 3 }, lst[{ 2, 3 }])
assert.are.same({ 1, 2, 3 }, lst[{ nil, 3 }])
