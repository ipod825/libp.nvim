Class FilePreviewBuffer

Module: libp.ui.FilePreviewBuffer

FilePreviewBuffer class.

Loading read-only file content to vim buffer asynchronously. There's also FileBuffer that creates modifiable buffer. Both FilePreviewBuffer and FileBuffer exists for loading file content into a floating Window asynchronously:

local ui = require("libp.ui")
require("plenary.async").void(function()
    local buf = ui.FilePreviewBuffer("file_to_load")
    local win = ui.Window(buf)
    win:open()
end)()

Parameters:

  • filename string The file name to load content from.
  • opts
    • listed boolean Whether the vim buffer is listed (see :help nvim_create_buf). (default false)
    • scratch boolean Whether the vim buffer is scratch. (see :help nvim_create_buf). (default true)
    • read_buffer_size number number of bytes to read in batch. This is used for performance tuning. (default 5000)
    • bo table buffer options to be set. See :help vim.bo. (default {})
