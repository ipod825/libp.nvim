Class OrderedDict

Module: libp.datatype.OrderedDict

Dict class that keeps the key order as they were inserted.

OrderedDict:data (d)
Returns the managed dict. The returned dict does not maintain the key insertion key order. Also, modifying the returned dict could make the maintained order stale. So make a deep copy if that is desired.

Parameters:

  • d OrderedDict The OrderedDict instance

Returns:

    table

Usage:

    local d = OrderedDict()
d.b = 1
d.a = 2
assert.are.same({ a = 2, b = 1 }, OrderedDict.data(d))
